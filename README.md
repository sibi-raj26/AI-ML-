# AI-ML- Repository

This repository contains my Artificial Intelligence (AI) and Machine Learning (ML) projects developed using Python and Google Colab. These projects demonstrate various machine learning algorithms, deep learning models, computer vision, and natural language processing techniques.

## Repository Contents

- 🌸 Iris Flower Prediction
- 🩺 Cancer Prediction
- 💰 School Fees Prediction
- 🤖 Rule-Based Chatbot
- 🧠 RNN Model
- 🖼️ AI Image Prediction using Transformers & YOLO

## Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Scikit-learn
- TensorFlow / Keras
- PyTorch
- OpenCV
- Ultralytics YOLOv8
- Hugging Face Transformers
- Matplotlib

## Project Descriptions

### 1. Iris Flower Prediction
Predicts the species of an iris flower using machine learning classification algorithms.

### 2. Cancer Prediction
A machine learning model that predicts whether a tumor is benign or malignant.

### 3. School Fees Prediction
Uses regression algorithms to predict school fees based on available features.

### 4. Rule-Based Chatbot
A simple chatbot that responds using predefined rules and conditions.

### 5. RNN Model
Implementation of a Recurrent Neural Network (RNN) for sequence learning tasks.

### 6. AI Image Prediction
Uses YOLOv8 for object detection and Hugging Face Transformers for image understanding.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sibi-raj26/AI-ML-.git
```

2. Navigate to the project folder

```bash
cd AI-ML-
```

3. Open any notebook in Google Colab or Jupyter Notebook.

4. Install the required packages if needed.

5. Run the notebook cells.
